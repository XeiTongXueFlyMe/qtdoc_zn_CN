StateMachineLoader QML Type

Dynamically loads an SCXML document and instantiates the state machine. More...

Detailed Description

Property Documentation

source : string

The url of the SCXML document to load. Only synchronously accessible URLs are supported.


stateMachine : QObject

The state machine instance.


Signal Documentation

sourceChanged()

This signal is emitted when the user changes the source URL for the SCXML document.


stateMachineChanged()

This signal is emitted when the stateMachine property changes. That is, when a new state machine is loaded or when the old one becomes invalid.
