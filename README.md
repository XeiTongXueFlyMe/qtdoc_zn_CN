# qtdoc_zn_CN
qt文档中文翻译项目

项目名称：**qt文档中文翻译项目**

开始时间：**2017年2月8日**

项目愿景：让中国人可以轻松看懂qt中文文档，熟练上手qt和C++.

翻译qt版本：5.8.0

翻译源文件：qt自带的html英文文件


翻译原则：
1.先熟悉通用术语翻译对照表；
2.了解模块翻译记录表；
3.加入感兴趣的翻译小组；
4.提交翻译好的中文html文件；
